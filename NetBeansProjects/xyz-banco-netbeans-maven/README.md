# XYZ Banco Console (Java 17, Maven)

## Abrir en NetBeans
1. Descomprime el ZIP.
2. En NetBeans: **File → Open Project...** y selecciona la carpeta que contiene `pom.xml`.
3. Clic derecho al proyecto → **Run**. O por consola: `mvn -q exec:java`.

## Estructura
- `src/main/java/com/xyz/banco/*`: código fuente.
- Clase principal: `com.xyz.banco.Main`.

Requisitos: JDK 17+ y Maven.
