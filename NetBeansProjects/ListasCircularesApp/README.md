# ListasCircularesApp (Java 17, Apache NetBeans 17)
Proyecto educativo con dos implementaciones:
- `listacircularsimple`: Lista circular simplemente enlazada.
- `listacirculardoble`: Lista circular doblemente enlazada con recorrido inverso.

## Cómo ejecutar en NetBeans
1. Archivo -> Abrir proyecto -> seleccionar carpeta `ListasCircularesApp`.
2. Para ejecutar una versión:
   - Click derecho en `Main.java` del paquete que quieras y elige **Run File**,
     o ajusta `main.class` temporalmente en `nbproject/project.properties`.
3. Java 17 requerido (configurado).
