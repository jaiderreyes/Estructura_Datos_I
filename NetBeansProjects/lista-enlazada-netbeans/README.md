# Lista Enlazada Simple (Maven + NetBeans)

Proyecto Java (Maven) con una implementación didáctica de **lista enlazada simple** y **menú interactivo por consola**.

## Requisitos
- Java 17+
- Maven 3.8+
- NetBeans (recomendado) o tu IDE favorito

## Ejecutar
```bash
mvn -q compile exec:java -Dexec.mainClass="com.ejemplo.lista.Main"
```

En NetBeans: clic derecho en el proyecto → **Run**.

## Operaciones incluidas
- Insertar al final
- Insertar al inicio
- Buscar
- Eliminar
- Mostrar
- Tamaño de la lista
