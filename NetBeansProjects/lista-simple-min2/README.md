# Lista Enlazada Simple - Versión Minimal (Maven + NetBeans)

Proyecto mínimo para enseñar lista enlazada simple con **dos opciones**: insertar y mostrar.

## Ejecutar con Maven
```bash
mvn -q compile exec:java -Dexec.mainClass="com.ejemplo.min.Main"
```

## Ejecutar con NetBeans
Abrir el proyecto y presionar **Run**.
